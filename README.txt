PROJETO LOGIN PARA EMPRESAS OU SITES

Projeto pessoal feito para simular um sistema de login para empresas ou sites. 
Nele é possível adicionar e ver os usuários além de um sistema de sessão para manter o usuário logado enquanto a página estiver aberta.

TECNOLOGIAS UTILIZADAS
	HTML
	CSS
	JAVASCRIPT
	PHP
	BANCO DE DADOS MYSQL
